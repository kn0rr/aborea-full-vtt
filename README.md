# ABOREA V7 (inoffiziell)

Dieses Paket enthält:
- separate Sheets für `character`, `npc`, `creature`
- Sidebar-Packs unter `packs/`
- Drag-and-drop auf Actors für alle Item-Typen
- automatische Anwendung von Volk und Beruf
- `_source`-Dokumente für Völker, Berufe, Waffen, Rüstungen, Zauber, Wunder und Kreaturen

Hinweis: Die Packs liegen als Foundry `_source`-Dokumente vor. Bei Nutzung in Foundry werden die Pack-Indizes aus den Quelldateien aufgebaut.
